# Internationalization (i18n) Skill

管理项目的多语言支持，包括英文(en)和中文(zh)。

## 当前支持的语言

| 语言 | 代码 | 路径 |
|------|------|------|
| English | en | /en/ |
| 中文 | zh | /zh/ |

## 添加新语言

### 1. 更新配置
编辑 `lib/i18n/config.ts`:
```typescript
export const locales = ['en', 'zh', 'nb', 'de'] as const;
export type Locale = (typeof locales)[number];
```

### 2. 更新字典加载
编辑 `lib/i18n/getDictionary.ts`:
```typescript
const dictionaries: Record<Locale, () => Promise<any>> = {
  en: () => import('@/locales/en/common.json'),
  zh: () => import('@/locales/zh/common.json'),
  nb: () => import('@/locales/nb/common.json'),  // 添加新语言
  de: () => import('@/locales/de/common.json'),  // 添加新语言
};
```

### 3. 更新语言切换器
编辑 `components/LocaleSwitcher.tsx`，添加新的 `DropdownMenuItem`。

### 4. 创建语言包
在 `locales/` 目录下创建新文件夹和 `common.json`:
```
locales/
├── nb/
│   └── common.json  # 挪威语
└── de/
    └── common.json  # 德语
```

### 5. 复制并翻译
复制 `locales/en/common.json` 的结构到新文件，然后翻译内容。

## 国际化页面结构

```
app/
├── (locale)/           # 多语言路由组
│   ├── layout.tsx      # 语言布局(加载字典)
│   └── [locale]/
│       └── page.tsx    # 首页
├── page.tsx            # 根路径重定向
└── liveAnalyser/           # 仍需国际化
    └── page.tsx
```

## 在页面中使用字典

```typescript
// app/(locale)/[locale]/page.tsx
export default async function Page({ params }: { params: { locale: string } }) {
  const dict = await getDictionary(params.locale as Locale);

  return (
    <h1>{dict.hero.title}</h1>
    <p>{dict.hero.description}</p>
  );
}
```
